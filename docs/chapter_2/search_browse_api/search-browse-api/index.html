Understanding Search & Browse API

Search and Browse API

After product catalog is indexed, client can use Search and Browse API to power their search and category pages. Indexed catalog can be viewed through search api using ?q=* query parameter

Consider the catalog at below link

https://search.unbxd.io/ad547a90aeeefe153a23ca942148fbcb/bonmarche-co-uk-stage3861566995969/search?q=*

Take a look at its correspondng website

Search API Usecase

Search Query

Search for particular product by typing query in the search box

https://search.unbxd.com/search?q=tshirt

Search Filter Query

Filtering the search api results based on the given product attribute

https://search.unbxd.com/search?q=tshirt&filter=size_uFilter:8

Search Multiple Filters ( AND Join )

Filtering the search api results based on multiple filters

https://search.unbxd.com/search?q=tshirt&filter=size_uFilter:8&filter=color_uFilter:Blue

Search Multiple Filters ( OR Join )

Filtering the search api results based on multiple filters joined by OR condition

https://search.unbxd.com/search?q=tshirt&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)

OR Join is not supported between two different filters

Search Multiple Filters ( Range Filters )

https://search.unbxd.com/search?q=tshirt&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)&filter=price_uFilter:[10 TO 50]

Search on category filters

Filtering search results based on the category they belong to

https://search.unbxd.com/search?q=tshirt&category-filter:"Home>Womens"

Search Sorting

https://search.unbxd.com/search?q=tshirt&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)&filter=price_uFilter:[10 TO 50]&sort=salesrank asc

Search facets Creation

https://search.unbxd.com/search?q=tshirt&facet=true&facet.field=size

It will create facets on field size. Facets can also be created and published through console

Search Multile facets creation

https://search.unbxd.com/search?q=tshirt&facet=true&facet.field=size&facet.field=color

It will create facets on field size and color. Facets can also be created and published through console

Seach Multilevel Category facets creation

https://search.unbxd.com/search?q=tshirt&facet=true&facet.field=categoryPath

Note, multi level category facets is created on categoryPath field. It is unbxd defined attribute which contains all category paths a product belongs to.

Browse API Usecase

Browse API

Fetch all products on a particular category

https://search.unbxd.com/search?p=categoryPath:"Home>Women"

If categoryPathId is known then browse API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID"

Browse Filter Query

Filtering the browse api results based on the given product attribute

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN"&filter=size_uFilter:8

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID"&filter=size_uFilter:8

Browse Multiple Filters ( AND Join )

Filtering the browse api results based on multiple filters

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN"&filter=size_uFilter:8&filter=color_uFilter:Blue

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID"&filter=size_uFilter:8&filter=color_uFilter:Blue

Browse Multiple Filters ( OR Join )

Filtering the browse api results based on multiple filters joined by OR condition

https://search.unbxd.com/p=categoryPath:"HOME>WOMEN"&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)

If categoryPathId is known then API would be

https://search.unbxd.com/p=categoryPathId:"HOMEID>WOMENID"&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)

OR Join is not supported between two different filters

Browse Multiple Filters ( Range Filters )

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN"&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)&filter=price_uFilter:[10 TO 50]

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID"&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)&filter=price_uFilter:[10 TO 50]

Browse on category filters

Filtering browse results based on the category they belong to

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN>TOPS"&category-filter:"Home>Womens"

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID>TOPSID"&category-filter-id:"HomeID>WomensID"

Browse Sorting

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN>TOPS"&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)&filter=price_uFilter:[10 TO 50]&sort=salesrank asc

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID>TOPSID"&filter=size_uFilter:8&filter=color_uFilter:(Blue OR Grey)&filter=price_uFilter:[10 TO 50]&sort=salesrank asc

Search facets Creation

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN&facet=true&facet.field=size It will create facets on field size. Facets can also be created and published through console

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID&facet=true&facet.field=size

Search Multile facets creation

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN&facet=true&facet.field=size&facet.field=color It will create facets on field size and color. Facets can also be created and published through console

If categoryPathId is known then API would be

https://search.unbxd.com/search?p=categoryPathId:"HOMEID>WOMENID&facet=true&facet.field=size&facet.field=color

Seach Multilevel Category facets creation

https://search.unbxd.com/search?p=categoryPath:"HOME>WOMEN&facet=true&facet.field=categoryPath

Note, multi level category facets is created on categoryPath field. It is unbxd defined attribute which contains all category paths a product belongs to.
